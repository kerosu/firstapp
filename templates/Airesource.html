python 函數與方法快速理解

以下是粗淺的判斷方式 首先 函數和方法都是"函數!" 會稱作的"函數"的如: len,max,min,sum,int......... 即所謂的內建函式 可用在各個地方ex:len((a,b,c,d)) len([a,b,c,d]) 可以看出len函數在tuple、list 都適用 再來看所謂的"方法": 第一類:通常你可能需要 import ( ) ；( ) 可以是math、random 、numpy math.sin(x) math.gcd(x,y) sin() 和 gcd() 函數只在math的模組下能被叫用 ，此時我們稱其為方法

第二類:我們來看看list 與 tuple list.append(x) list.remove(x) 中的append、remove 都是屬於list 類別下的方法 若是你將append 應用到tuple t1=(1,2,3,4) t1.append(5) 你會得到:AttributeError: 'tuple' object has no attribute 'append' 也就是append 是在特定類別才能使用的函數，我們通常稱其為方法!!!fe cray tumblr retro.
